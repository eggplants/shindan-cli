## Usage

## CLI

```shellsession
$ shindan -h
usage: shindan [-h] [-V] ID NAME

ShindanMaker (https://shindanmaker.com) CLI

positional arguments:
  ID             shindan page id
  NAME           shindan name

optional arguments:
  -h, --help     show this help message and exit
  -V, --version  show program's version number and exit
```
